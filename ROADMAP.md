# Roadmap

## Near Term

- Keep GitHub release downloads as the primary install path for end users
- Improve release notes so each asset clearly states the target distro or install method
- Verify packaged installs on more COSMIC-based Debian and Ubuntu systems

## Product

- Add audio and microphone capture support
- Add a native floating recording overlay with timer and stop controls
- Replace the current `slurp`-based selection helper with a native Taffy region selector

## Platform

- Improve portal capability diagnostics so users can tell why shortcuts or capture flows are limited
- Revisit global shortcut support as COSMIC portal support evolves
- Continue polishing desktop integration for launcher, icon, and taskbar consistency

## Packaging

- Keep AppImage and Debian releases polished and easy to install
- Refine RPM dependency metadata for specific RPM-based targets before recommending it broadly
- Add more release automation once the packaging formats stabilize
